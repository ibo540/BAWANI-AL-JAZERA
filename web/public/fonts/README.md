# Fonts

Place **TSTaric.woff2** (TS Taric Arabic font) here for the Arabic locale.

The site uses:
- **English:** Montserrat (Google Font)
- **Arabic:** TS Taric (local file: `TSTaric.woff2`)

If you have a different filename (e.g. `TSTaric-Regular.woff2`), update the path in `app/globals.css` inside the `@font-face` rule.
